## 1. 遊戲概念與定位

### 1.1 一句話介紹
> 「你是清廷書記，從 1860 年起，三十年間踏遍九州——每一年你決定去哪、跟誰、做什麼，親眼見證洋務運動的興衰。」

### 1.2 核心體驗
- **空間自由**：玩家在 10 個城市間自由移動
- **時間壓力**：歲月推進不可逆，60 歲強制結算
- **真實邂逅**：每個城市有獨立故事與歷史人物
- **後果累積**：見證 / 錯過的事件決定通關評價
- **多周目價值**：4 條路線 + 隱藏事件，一局看不完

### 1.3 與劇情版的關係
| | 劇情版《自強之路》 | 探索版《自強三十年》 |
|---|---|---|
| **玩家身份** | 隨官員的書記（觀察者） | 自主行動的書記（主角） |
| **時間** | 線性 4 章，被劇情推進 | 30 回合，自己掌握節奏 |
| **空間** | 14 個固定場景 | 10 個城市自由移動 |
| **失敗** | 無 | 有：60 歲退休 / 信用歸零 |
| **內容** | 共用同一份 data.js | 共用同一份 data.js |
| **適合場景** | 第一次接觸 / 課堂帶領 | 第二次重玩 / 課後深化 |

### 1.4 設計信條（不可妥協）
1. **沒有廢城市**：每個城市都有獨立精彩內容
2. **沒有廢回合**：每個玩家行動都有意義
3. **DSE 內容 100% 可觸及**：鐵釘事件 + 補課小卡確保覆蓋
4. **歷史真實優先**：任何遊戲機制不得扭曲史實
5. **單局 30–45 分鐘**：適合課堂使用

## 2. 核心循環

### 2.1 回合流程圖

```
┌─────────────────────────────────────────┐
│ 1. 顯示當前年份、季節、玩家狀態         │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│ 2. 檢查：本年是否觸發鐵釘事件？         │
│   是 → 強制跳轉到事件地點，播放劇情     │
│   否 → 進入自由選擇                     │
└───────────────┬─────────────────────────┘
                ↓ (自由回合)
┌─────────────────────────────────────────┐
│ 3. 顯示地圖：當前所在城市 + 可前往城市  │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│ 4. 玩家選擇：                            │
│   (a) 停留當前城市，進行活動            │
│   (b) 移動到另一城市（消耗時間 + 旅費） │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│ 5. 執行行動：                            │
│   • 跟隨官員（觸發對話）                │
│   • 探查地點（收集 evidence）           │
│   • 寫私函（鞏固學習）                  │
│   • 處理朝廷雜務（補俸祿/信用）         │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│ 6. 結算：時間推進、資源變動、事件記錄    │
└───────────────┬─────────────────────────┘
                ↓
        (回合 30 後結算結局)
```

## 3. 時間軸：混合制回合系統

### 3.1 時間架構

**起點**：1860 年（玩家 25 歲入仕）
**終點**：1895 年（甲午戰後）或 60 歲退休
**總跨度**：35 年
**回合數**：約 30 回合

### 3.2 回合單位設計

採用**混合制**：

| 階段 | 時間單位 | 用途 |
|---|---|---|
| **史實密集期** | 1 年 = 1 回合 | 重大事件密集年份（1860, 1861, 1872, 1881, 1888, 1894, 1895）|
| **自由探索期** | 1 年 = 1 回合 | 中間年份，玩家自由規劃 |
| **時間跳躍** | 自動快進 | 無重大事件的年份自動跳過（如 1869–1871）|

**示例時間軸**：
```
1860 ─ 1861 ─ 1862 ─ [1863–1864 自由] ─ 1865 ─ [1866–1867 自由]
─ 1868 ─ [1869–1871 跳躍] ─ 1872 ─ [1873–1874 自由] ─ 1875 ─
[1876–1880 自由（5回合）] ─ 1881 ─ [1882–1887 自由（5回合）] ─
1888 ─ [1889–1893 自由（4回合）] ─ 1894 ─ 1895
```

共約 30 個回合。

### 3.3 季節機制（增強真實感）

每年分**春、夏、秋、冬** 4 季，但回合計算以「年」為單位。
- **冬季**：海運受阻，海路移動 +50% 時間
- **夏季**：南方有疫病風險，去廣州/福州有 10% 健康損耗
- **秋季**：科考季，去北京可獲額外朝廷信用
- **春季**：標準狀態

季節由玩家進入該年自動定（春→夏→秋→冬循環），有助於增加策略性。

## 4. 12 個鐵釘事件清單

這 12 個事件**無論玩家在哪都會發生**，自動傳送玩家到事件地點觀看核心劇情（每段 3–5 分鐘）。

| # | 年份 | 地點 | 事件名稱 | DSE 核心 |
|---|---|---|---|---|
| **1** | 1860 | 北京 | 英法聯軍入京、火燒圓明園 | 列強衝擊、清廷無力抵禦 |
| **2** | 1861 | 北京 | 總理衙門設立、奕訢辭訓 | 近代外交制度開端 |
| **3** | 1862 | 北京 | 同文館成立 | 西學教育起步 |
| **4** | 1865 | 上海 | 江南製造總局創辦 | 軍事工業核心 |
| **5** | 1866 | 福州 | 船政局創辦、船政學堂 | 海軍與技術教育 |
| **6** | 1872 | 上海/香港 | 首批留美學童出發 | 人才培育、容閎 |
| **7** | 1874 | 福州 | 日本侵台、台灣防務 | 由強轉富之始 |
| **8** | 1881 | 上海 | 留美學童撤回 | 守舊派反撲、洋務局限 |
| **9** | 1885 | 福州 | 馬尾海戰、中法戰爭 | 海軍初遇實戰 |
| **10** | 1888 | 威海衛 | 北洋艦隊正式成軍 | 洋務高峰 |
| **11** | 1894 | 黃海/威海衛 | 甲午海戰、黃海大戰 | 洋務挫敗 |
| **12** | 1895 | 馬關 | 馬關條約簽訂 | 洋務終結 |

### 4.1 鐵釘事件的觸發設計

```
當回合進入年份 X：
  if X in [1860, 1861, 1862, 1865, 1866, 1872, 1874, 1881, 1885, 1888, 1894, 1895]:
    顯示「歷史時刻」過場（黑屏 + 年份大字）
    玩家自動傳送到事件地點
    播放該事件的核心對話（3-5 分鐘）
    解鎖該事件對應的核心 evidence
    玩家獲得「見證者」標記
  else:
    進入自由回合
```

### 4.2 鐵釘事件 vs 自由事件的內容劃分

| 類型 | 數量 | 來源 |
|---|---|---|
| **鐵釘事件** | 12 | 直接從劇情版 14 場景的核心對話節錄改編 |
| **自由事件** | ~30 | 部分來自劇情版次要對話 + 新增 |

### 5.2 城市詳細卡

#### 北京（政治中心）
- **主題**：朝廷政治、外交、思想之爭
- **核心人物**：奕訢、倭仁、徐桐、同治帝、光緒帝
- **可觸發事件**：
  - 總理衙門設立辯論（1861）★
  - 同文館之爭（1867）★
  - 留美學童撤回廷議（1881）★
  - 慈禧挪用海軍經費（1888 後）
  - 各派閣老論爭（年年可觸發）
- **可探查地點**：紫禁城、同文館、總理衙門、頤和園
- **常駐 NPC**：奕訢（1861–1898）、倭仁（–1871）、翁同龢、徐桐
- **特色機制**：每次到訪有機會獲得「朝廷信用」（聽訓得分）

#### 廣州（通商口岸 / 鴉片戰爭遺址）
- **主題**：對外貿易、鴉片戰爭遺跡、十三行
- **核心人物**：兩廣總督（耆英→葉名琛→...→張之洞 1884）
- **可觸發事件**：
  - 探訪虎門炮台殘垣（任時段）
  - 十三行與洋商交涉（1860–1870）
  - 張之洞督粵改革（1884–1889）★
- **可探查地點**：虎門炮台、十三行舊址、總督府
- **特色機制**：是「列強衝擊」evidence 主要來源

#### 上海（工商業樞紐）
- **主題**：江南製造、機器織布、招商局、買辦
- **核心人物**：李鴻章（駐節）、唐廷樞、容閎、傅蘭雅、徐壽
- **可觸發事件**：
  - 江南製造總局創辦（1865）★鐵釘
  - 譯書館啟動（1867）
  - 容閎籌劃留美（1870–1872）★
  - 留學生啟程（1872）★鐵釘
  - 機器織布局開機（1890）
  - 留學生撤回（1881）★鐵釘
- **可探查地點**：江南製造總局、譯書館、招商局總部、外灘洋行
- **特色機制**：「求富」evidence 主要來源；可賺取俸祿（協助商務）

#### 南京（條約之城）
- **主題**：南京條約遺跡、太平天國餘響、湘軍據點
- **核心人物**：曾國藩（1861–1872 兩江總督）、沈葆楨
- **可觸發事件**：
  - 安慶內軍械所視察（1861–1864）
  - 太平天國平定後重建（1864 後）
  - 曾國藩去世（1872）
- **可探查地點**：總督衙門、靜海寺（條約簽署地）、安慶軍械所遺址
- **特色機制**：與曾國藩的關係決定 1872 後是否被沈葆楨接見

#### 福州（海軍搖籃）
- **主題**：船政局、船政學堂、海軍技術
- **核心人物**：左宗棠（1866 創辦）、沈葆楨（接任）、嚴復（學員）
- **可觸發事件**：
  - 船政局創辦（1866）★鐵釘
  - 船政學堂開學（1867）
  - 第一艘國造輪船下水（1869）
  - 嚴復畢業赴英（1877）
  - 日本侵台（1874）★鐵釘
  - 馬尾海戰（1885）★鐵釘
- **可探查地點**：船政局、船政學堂、馬尾港
- **特色機制**：「軍事工業」與「人才培育」evidence 集中地

#### 天津（北洋根據地）
- **主題**：北洋大臣府、武備學堂、大沽炮台
- **核心人物**：李鴻章（1870–1895 直隸總督）、丁汝昌、漢納根
- **可觸發事件**：
  - 李鴻章接任直隸總督（1870）
  - 武備學堂創辦（1885）
  - 北洋艦隊成軍籌備（1885–1888）
  - 漢納根任顧問（年年）
- **可探查地點**：北洋大臣府、武備學堂、大沽炮台、海光寺機器局
- **特色機制**：跟隨李鴻章路線的主要據點

#### 武漢 / 漢陽（重工業中心）
- **主題**：漢陽鐵廠、湖廣督署、張之洞改革
- **核心人物**：張之洞（1889–1907 湖廣總督）、盛宣懷
- **可觸發事件**：
  - 漢陽鐵廠籌建（1890）★
  - 漢陽鐵廠開爐（1894）
  - 鐵廠技術困局（1894 後）
  - 湖北織布局（1892）
- **可探查地點**：漢陽鐵廠、兩湖書院、湖廣督署
- **特色機制**：「重工業」evidence 來源；1889 後才解鎖

#### 威海衛（海軍基地）
- **主題**：北洋艦隊、甲午戰場
- **核心人物**：丁汝昌、劉公島駐軍、定遠/鎮遠艦長
- **可觸發事件**：
  - 北洋艦隊閱兵（1888）★鐵釘
  - 甲午海戰前夕（1894 春）
  - 黃海海戰（1894 秋）★鐵釘
  - 威海衛陷落、丁汝昌殉國（1895 春）
- **可探查地點**：劉公島、定遠艦、北洋海軍提督府
- **特色機制**：1895 後封閉（已陷落）

#### 開平（煤礦重鎮）
- **主題**：開平煤礦、唐胥鐵路
- **核心人物**：唐廷樞、礦工
- **可觸發事件**：
  - 開平煤礦籌辦（1877）
  - 唐胥鐵路通車（1881）
  - 礦工抗爭事件（隨機）
- **可探查地點**：開平礦務局、唐胥鐵路、唐廷樞官邸
- **特色機制**：「礦冶/鐵路」evidence 來源

#### 香港（中介窗口）
- **主題**：容閎據點、留學籌劃、西學書籍
- **核心人物**：容閎（1854–1872 多次經港）、留美學童家屬
- **可觸發事件**：
  - 容閎拜訪曾國藩前途經（1863）
  - 留美學童家屬集結（1872）
  - 撤回學童經香港（1881）
- **可探查地點**：聖約翰座堂、傳教士書館、容閎寓所
- **特色機制**：解鎖「容閎路線」的關鍵城市；可購入西學書籍（+見識）

## 6. 資源系統

### 6.1 四項資源

| 資源 | 範圍 | 補充方式 | 消耗 / 損失方式 |
|---|---|---|---|
| **歲月（年齡）** | 25 → 60 | 不可補充 | 自動推進 |
| **俸祿（銀兩）** | 0–100 | 每年自動 +20；協助商務 +5；朝廷賞賜 +10 | 移動消耗（按距離 5–20）；探查 −5 |
| **朝廷信用** | 0–10（從 5 起步） | 完成上級交辦任務 +1；參與朝廷活動 +1 | 自作主張 −1；錯過鐵釘事件 −1；違背官員意願 −2 |
| **見識（DSE 內化度）** | 0–100 | 收集 evidence +1–3；完成事件 +5；寫私函 +3 | 不會減少 |

### 6.3 資源的策略張力

- **俸祿不足**：無法移動，被困一城（要去處理朝廷雜務補俸祿）
- **信用過低（≤2）**：可能被外放閒職 = Game Over（提前退休）
- **歲月推進**：60 歲強制結算
- **見識**：通關評分主要依據，不影響存活

## 7. 4 條路線設計

### 7.1 路線選擇時機

開局後第一個回合（1860）由玩家選擇路線，路線決定：
- 初始城市
- 初始恩主（官員）
- 該年的特定事件可觸發
- 通關評語

### 7.2 路線詳細設計

#### 路線 A：跟隨李鴻章（1860 → 1895）
- **起點**：1860 年，上海，加入剛升任的李鴻章幕府
- **路徑**：上海（淮軍時期）→ 江蘇巡撫府 → 1870 後天津（直隸總督）
- **主軸**：軍事工業 + 北洋艦隊
- **必經事件**：江南製造（1865）→ 武備學堂（1885）→ 北洋成軍（1888）→ 甲午（1894–95）
- **特色機制**：每年自動有「李鴻章召見」事件，可選參與或自由行動
- **通關特典**：解鎖李鴻章視角的私函系列
- **歷史 framing**：「你親歷洋務派旗手的興衰」

#### 路線 B：跟隨容閎（1860 → 1881）
- **起點**：1860 年，香港，剛從耶魯回國的容閎
- **路徑**：香港 → 上海 → 南京（拜訪曾國藩 1863）→ 北京 → 1872 親送學童赴美
- **主軸**：人才培育 + 西學東漸
- **必經事件**：江南製造譯書館（1867）→ 留美學童啟程（1872）→ 撤回（1881）
- **特殊結束**：1881 後路線淡出（容閎事業中挫）
- **特色機制**：可獲得獨家「留學生通訊」evidence
- **通關特典**：「容閎私函集」
- **歷史 framing**：「你陪一個失敗的夢想家走完他的高峰」

#### 路線 C：跟隨奕訢（1860 → 1895）
- **起點**：1860 年，北京，剛留京談判的奕訢
- **路徑**：以北京為主，偶爾外派視察
- **主軸**：朝廷政治 + 外交 + 思想之爭
- **必經事件**：總理衙門（1861）→ 同文館之爭（1867）→ 留學廷議（1881）→ 退隱（1884，甲申易樞）
- **轉折**：1884 奕訢失勢，路線轉「李鴻章餘暉」或「自由書記」
- **特色機制**：朝廷信用易得，可介入更多政策決定
- **通關特典**：「總理衙門檔案」
- **歷史 framing**：「你站在權力核心，看見每個改革都被妥協」

#### 路線 D：自由書記（1860 → 1895）
- **起點**：1860 年，北京，無特定恩主的編外書記
- **路徑**：完全玩家自主
- **主軸**：無預設，由玩家行動決定
- **特色機制**：
  - 沒有「自動召見」事件，玩家完全自由
  - 可同時為多位官員效力
  - 必須自己經營朝廷信用（較難）
- **通關特典**：依玩家走過的城市給予不同評語
- **歷史 framing**：「你不是任何派系的人，你只是一個想看清這個時代的人」
- **難度**：高（無路線指引）

### 7.3 路線完成度標準

| 路線 | 通關條件 |
|---|---|
| A 李鴻章 | 活到 1895 + 見證 ≥10 個鐵釘事件 |
| B 容閎 | 完成 1881 撤回事件（其後可繼續但已結束主線） |
| C 奕訢 | 活到 1895 + 朝廷信用始終 ≥3 |
| D 自由書記 | 活到 1895 + 見識 ≥50 |

完成度不影響結局，影響通關評語與成就解鎖。

## 8. 失敗與結算機制

### 8.1 失敗條件

**條件 1：60 歲強制退休（時間結算）**
- 1895 年（玩家 60 歲）強制結算
- 不算失敗，依見識評分

**條件 2：朝廷信用歸零（政治失寵）**
- 信用降至 0 = 被外放閒職、提前退休
- 結算畫面：「你失寵了。從此不再被召見……」
- 仍會給見識評分，但顯示「未竟之路」

**條件 3：俸祿長期歸零（生計困頓）**
- 連續 3 回合俸祿 = 0 = 不得不告老還鄉
- 結算畫面：「家中老母來信，言生計困頓……」
- 同樣給見識評分
